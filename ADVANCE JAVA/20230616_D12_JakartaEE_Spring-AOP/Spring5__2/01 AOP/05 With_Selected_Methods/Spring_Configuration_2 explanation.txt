
<bean id="productPointcut" 
                class="org.springframework.aop.support.NameMatchMethodPointcut">
		<property name="mappedName" value="*" />
</bean>

	above tag means that

		productPointcut  -  represents all the methods of target class ( because of "*" )


<bean id="productAdvisor" 
                 class="org.springframework.aop.support.DefaultPointcutAdvisor">
		<property name="pointcut" ref="productPointcut" />
		<property name="advice" ref="beforeCall" />
	</bean>


	above tag means

		advisor says that
			productPointcut ( all the methods ) --   "before" advice
		i.e.  "before" advice should be applied to all the methods of target class.
