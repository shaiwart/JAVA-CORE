While running application if u get a warning, "No appenders could be found for logger(log4j)?", perform following steps:


Make sure your project is open in Eclipse, then click on the "Run" menu at the top of Eclipse and click on the following:

Run
Run Configurations
Classpath (tab)
User Entries
Advanced (button on the right)
Add Folders
then navigate to the folder that contains your log4j.properties file i.e "src"
Apply
Run
The error message should no longer appear.
